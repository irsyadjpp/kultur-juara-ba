# **App Name**: BCC 2026 Landing

## Core Features:

- Header Navigation: Sticky top navigation bar with links to Home, Jadwal (Ayo App), and Contact.
- Hero Section: Prominent section with title, subtext, and a red 'DAFTAR SEKARANG' CTA button redirecting to Ayo Indonesia registration.
- Live Info Widget: Display real-time data from Firestore: Slot Terisi and Countdown to June 13, 2026.
- Value Proposition Cards: Highlight key USPs: Anti-Sandbagging System, Total Hadiah (Rp 42 Juta+), and WBD Celebration.
- Sponsor & Partner Showcase: Display logos of main sponsors (Ayo Indonesia, Bank BJB) and operational partners (Flypower, Pocari Sweat, Bebas Cedera).
- Footer Information: Contact details, link to Technical Handbook (PDF), and Integrity Commitment clause.
- SEO Optimization: Implement basic SEO metadata for title and description.

## Style Guidelines:

- Primary color: Sporty red (#D2232A) for energy and action.
- Secondary color: Navy blue (#003366) for a formal and reliable feel.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Header font: 'Montserrat' (sans-serif) for a modern and sporty look. Note: currently only Google Fonts are supported.
- Body text font: 'Merriweather' (serif) for readability. Note: currently only Google Fonts are supported.
- Ensure fast loading times and compatibility with iOS and Android devices.
- Use sports-themed icons to enhance visual appeal and convey key information.
